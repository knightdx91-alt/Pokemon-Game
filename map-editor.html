How to use

• Pick a tileset, set W/H, hit New.
• Click a metatile in the left palette, then paint on the map.
• Fill = bucket. Rect = drag a rectangle. Pick = eyedrop a tile.
• Collision mode: paint red = blocked, click again to clear.
• Export downloads <layout>.json + <map>.json.
• Import a layout JSON to keep editing it.
